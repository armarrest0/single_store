立即购买： 第一个数字为商品编号，第二个数字为购买数量，用竖线分割。例：232|1
购物车购买：第一个数字为购物车编号，第二个数字为购买数量，用竖线分割。多组用半角逗号分割，例：232|1,110|2 232商品购买1个，110商品购买2个
